Factor

[tex]\\[/tex]

If (d+7) is one of the factors of d³ + 343, what is the other factor?

[tex]\\[/tex]

Analyzing the polynomial, d³ + 343 is an example of a sum of two cubes.

[tex]\\[/tex]

Formula:

x³ + y³ = (x + y)(x² - xy + y²)

where x is the value of the constant in the linear term and y is the constant itself.

[tex]\\[/tex]

Solution:

Given the first factor (d+7), x would be 1 and y would be 7. If values are substituted–

[tex]\\[/tex]

= x³ + y³ = (x + y)(x² - xy + y²)

= d³ + 343 = (d + 7)(1) d² - ( 1 )( 7 ) d + (7)²

= d³ + 343 = (d + 7)( d² - 7d + 49)

[tex]\\[/tex]

Notice that one of the factor is (d+7), which is already given. The other factor is (d²-7d+49).

[tex]\\[/tex]

Answer:

≡ [tex]\begin{gathered} \boxed{ \large\boxed{\tt{d^{2}-7d+49 }}} \end{gathered}\\\\[/tex]

Letter D is the answer.
