To calculate the answer correctly, we need to clarify 2 points in the question:

“Php1, 575” contains a typo. The space is in the wrong place. It should be “Php 1,575.” That is, the total value of the coins is 1,575 pesos.
A simple clarification: X represents the number of 10-peso coins; Y represents the number of 5-peso coins.
1) X + Y = 207; total number of coins is 207, given

2) 10X + 5Y = 1,575; total value of coins is 1,575, given

3) X = 207 – Y; solve line 1 for X

4) 10(207 – Y) + 5Y = 1,575; substitute line 3 into line 4

5) 2070 – 10Y + 5Y = 1,575; expand left side

6) 2070 – 5Y = 1,575; simplify left side

7) 2070 = 1,575 + 5Y; add 5Y to each side

8) 5Y = 2070 – 1,575; switch sides on equal sign, subtract 1,575 from each side

9) 5Y = 495; resolve subtraction

10) Y = 99; divide each side by 5 to obtain value of Y. There are 99 5-peso coins

Obtain value of X

11) X + Y = 207; restate equation 1

12) X = 207 – Y; solve for X

13) X = 207 – 99; put in value of Y

14) X = 108; perform subtraction on right side. There are 108 10-peso coins

Check answer

28. a wallet containing only 5 peso and 10 peso coins, has 6 more 10 peso coins than 5 peso coins. the total value of the money is Php360. how many 5 peso coins and 10 peso coins are in the purse?​


Let x be the number of 5 peso coins.

Then, the number of 10 peso coins is x+6.

The total value of the coins is given by:

5x + 10(x+6) = 360

Simplifying this equation, we get:

15x + 60 = 360

15x = 300

x = 20

Therefore, the number of 5 peso coins is 20 and the number of 10 peso coins is x+6 = 26.

5 = 20

10 = 26

20 x 5 = 100

10 x 26 = 260

260 + 100 = 360
