4. a rectangular garden has a length 3 meters more than twice its width.if the area is 90 sqm, what is it's width and length​


Answer:

Assume that the garden forms a rectangle.

Let L = length and W = width

The area is A = LW given A = 90 m². So 90 = LW

The length is 3 more than twice it's width is written as L = 3 + 2W

Substitute the equation in terms of W:

90 = (3 + 2W)(W)

90 = 3W + 2W²

Arranging,

2W² + 3W - 90 = 0

Using quadratic formula, where a = 2, b = 3, c = -90

\begin{gathered}w = \frac{-b\pm\sqrt{-b^2-4ac} }{2a} \\w = \frac{-3\pm\sqrt{(-3)^2-4(2)(-90)} }{2(2)} \\\\w = \frac{-3\pm\sqrt{729} }{4} \\\\w = \frac{-3\pm27}{4} \\\\w = 6;w= -7.8\end{gathered}
