11. It is a polygon with eight sides. If all the sides and angles are equal,the measure of each angle is 135 degrees?


Octagon

Explanation:

Because the octagon is regular, all of its sides and angles are congruent. Thus, the measure of each angle is equal to the sum of its angles divided by 8. Therefore, each angle in the polygon has a measure of 1080/8 = 135 degrees.

30. A heptagon with interior angles measuring 117, 125, 133, 111, 135, 123, degrees​


Answer:

To find the perimeter of a heptagon with interior angles measuring 117, 125, 133, 111, 135, 123 degrees, we can use the formula:

Perimeter = (n x s)

where n is the number of sides and s is the length of each side.

For a heptagon, n = 7. To find the length of each side, we can use the fact that the sum of the interior angles of a polygon with n sides is given by the formula:

Sum of interior angles = (n - 2) x 180 degrees

So for a heptagon, we have:

Sum of interior angles = (7 - 2) x 180 = 900 degrees

We can use this sum to find the missing angle by subtracting the sum of the given angles from the total sum:

Missing angle = Sum of interior angles - Sum of given anglesMissing angle = 900 - (117 + 125 + 133 + 111 + 135 + 123) = 156 degrees

Since all sides of a regular heptagon are congruent, we can assume that the heptagon in this problem is regular. Therefore, we can use the fact that the sum of all interior angles in a regular heptagon is equal to (n-2) x 180 degrees, divided by the number of sides, to find the measure of each angle:

Each interior angle = [(n - 2) x 180] / nEach interior angle = [(7 - 2) x 180] / 7Each interior angle = 128.57 degrees (rounded to two decimal places)

Now, we can use the fact that the sum of all angles in a polygon is equal to (n-2) x 180 degrees to find the sum of the sides:

Sum of angles = (n-2) x 180 degreesSum of angles = (7-2) x 180 degreesSum of angles = 900 degrees

Since we know that the sum of the angles of a heptagon is 900 degrees, we can find the perimeter by dividing the sum of the sides by the length of each side:

Perimeter = (n x s) = (7 x s) = (Sum of angles) / (Each interior angle)Perimeter = (900 degrees) / (128.57 degrees) = 7.00

Therefore, the perimeter of the heptagon is 7 times the length of each side, which is not specified in the problem.
