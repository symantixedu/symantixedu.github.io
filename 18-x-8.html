3. What should be multiplied to (x + 2) to get a product of (x2 + 10x + 16) ? d. x - 18 a. X + 8 b. x - 8 c. X + 18​


Answer:

a. x + 8

Step-by-step explanation:

Let (x + 2) multiplied to N to get product of (x² + 10x + 16) so,

[tex] (x + 2)(N) = x {}^{2} + 10x + 16[/tex]

Factoring right side :

[tex](x + 2)(N) = x {}^{2} + 8x + 2x + 16[/tex]

[tex](x + 2)(N) = x(x + 8) + 2(x + 8)[/tex]

[tex]\cancel{(x+2)} \: (N) = \cancel{(x+2)} \: (x + 8)[/tex]

Cancel out (x + 2) from each side we get :

[tex] \: \:{\boxed{{{N=(x+8)}}}}[/tex]

Thus, (x + 8) should multiplied to (x + 2) to get a product of (x² + 10x + 16).

x + 3/8 - 3/8 = 18 - 3/8

x = 18 - 3/8

We can convert 18 to an improper fraction by multiplying the whole number (18) by the denominator of the fraction (8), and then adding the numerator of the fraction (0):

18 = 144/8

So we have:

x = 144/8 - 3/8

Simplifying the right-hand side by subtracting the numerators:

x = 141/8

Therefore, the solution is:

x = 141/8 or x = 17.625
