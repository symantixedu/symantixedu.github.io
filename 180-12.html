6. if -12 is one of the factors of -180, what is the other factor?​


Answer:

The other factor is 15.

Step-by-step explanation:

If two factors make up -180, and -12 is one of those factors, then the other factor is what we can demonstrate as x.

Solving for x:

[tex] - 12x = - 180 \\ \frac{ - 12x}{\blue{ - 12}} = \frac{ - 180}{\blue{ - 12}} \\ x = 15[/tex]

[tex]\blue{\large{\boxed{x=15}}}[/tex]

Conclusion:

The other factor, which we demonstrated as x, is 15.

Step-by-step explanation:

LCM(12, 18, 180) =

LCM(LCM(12, 18), 180)

Working from the innermost parentheses outward:

LCM(12,18) = (12 × 18) / GCF(12,18)

= (12 × 18) / 6

= 216 / 6

= 36

LCM(36,180) = (36 × 180) / GCF(36,180)

= (36 × 180) / 36

= 6480 / 36

= 180

Therefore,

LCM(12, 18, 180) = 180

Step-by-step explanation:

45° + 360° + X = 180°

First, we can simplify the left side by adding 45° and 360°:

405° + X = 180°

Next, we can isolate X by subtracting 405° from both sides:

X = 180° - 405°

X = -225°

To check, we can substitute X = -225° back into the original equation and see if both sides are equal:

45° + 360° - 225° = 180° (true)

37° + 91° + X = 180°

To solve for X, we can simplify the left side by adding 37° and 91°:

128° + X = 180°

Next, we can isolate X by subtracting 128° from both sides:

37° + 91° + 52° = 180° (true)

X + 80° = 180°

To solve for X, we can isolate X by subtracting 80° from both sides:
