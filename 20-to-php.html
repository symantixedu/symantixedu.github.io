19. Pls help me in my assignment..... Flor withdrew Php. 1780 from a bank. The bank teller gave her Php. 100 and Php.20 bills. If the number of Php.20 bills was 5 more than twice the number of Php.100 bills, how many Php.20 bills did she receive?


Answer:

Flor received 29  Php 20 bills

Step-by-step explanation:

Let x be the number of Php 100 bills

Let y be the number of Php 20 bills

We know that the number of Php 20 bills was more than twice the number of Php 100 bills, from that we can get this equation

y= 2x +5

So we try to get the number of bills

100x+20y=1780    

We multiply x and y to 100 and 20 because that is the denomination of 100 and 20 Peso bills

100x + 20y = 1780

100x + 20(2x +5) = 1780

100x + 40x +100= 1780

140x=1680

x= 12

So now we know that we have 12 Php 100 bills

We now get the number of Php 20 bills

y=2x+5

y=2(12)+5

y=29

We can check this by simply substituting the values of x and y to the equation

100x+20y=1780  

100(12)+20(29)=1780

1200+580=1780

1780=1780

Checking means it is correct
