Solving Functions

Function is a relation between certain sets of numbers, variables, or both. A function takes an input value, performs some process, then yields an output value. In f(x), the function f simply takes in input value x. Any value of x works in function as long as it is defined. This include fractions as well as whole numbers, both for positive and negative values.

= x² + x - 20
