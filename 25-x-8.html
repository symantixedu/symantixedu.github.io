21. If x:8= y:32 andx + y =25, what is y-x?​


Answer:

The answer is equal to 15, I think.

Answer:

y - x = 15,

x = 5, y = 20

Step-by-step explanation:

Think of ratios x:8 and y:32 as a fraction.

x:8 = y:32 ≈ [tex]\frac{x}{8}[/tex] = [tex]\frac{y}{32}[/tex]

Use one of different methods to solve systems of equations.

In this case, substitution.

{ [tex]\frac{x}{8}[/tex] + [tex]\frac{y}{32}[/tex]

{ x + y = 25

Start with first equation, [tex]\frac{x}{8}[/tex] + [tex]\frac{y}{32}[/tex].

To make it easier and clearer, we make the two like or similar fractions.

[tex]\frac{x}{8}[/tex] = [tex]\frac{4x}{32}[/tex]

We achieved this by simply multiplying 4 to both denominator and numerator. 4 to make the denominator 32.

This then means that 4x = y because the fractions now have the same denominators.

  To see this method even clearer, let us see it in another, more sensible scope.

  Since we are given [tex]\frac{4x}{32}[/tex] = [tex]\frac{y}{32}[/tex],

  we can then multiply 32 to both sides through MPE or

  Multiplication Property of Equality, so this becomes

    (32) [tex]\frac{4x}{32}[/tex] =  [tex]\frac{y}{32}[/tex](32)

  ≈ 4x =  [tex]\frac{32y}{32}[/tex].

  32 divided by 32 is one, since of course, any number divided by

  itself equates to 1.

  So we therefore get 4x = y.

Now we go back to our second equation x + y = 25.

Then we substitute x for y since we know that 4x = y.

Our equation then becomes x + 4x = 25

Then this becomes 5x = 25 since 4x + (1)x = 5x.

5x = 25

Then we multiply both sides by [tex]\frac{1}{5}[/tex], or in simple

terms, divide both sides by 5 to find x.

[tex]\frac{5x}{5}[/tex] =  [tex]\frac{25}{5}[/tex],

[tex]\frac{5x}{5}[/tex] is x, and [tex]\frac{25}{5}[/tex] is 5.

So now we know that x = 5.

To find y, we can get back to the equation we concluded - 4x = y.

4(5) = y

then y = 20.

To check, we can simplify the fractions we derived from the ratios.

Substitute x and y for their corresponding values.

[tex]\frac{5}{8}[/tex] + [tex]\frac{20}{32}[/tex]

We know that [tex]\frac{5}{8}[/tex] is in simplest term, so we focus on [tex]\frac{20}{32}[/tex].

To simplify, we reduce the fraction by dividing the numbers by a common factor. If possible, the greatest common factor to speed up the process.

The GCF of 20 and 32 is 4. So we divide both numerator and denominator by 4 to reduce the fraction to lowest terms.

[tex]\frac{20/4}{32/4}[/tex] is [tex]\frac{5}{8}[/tex]

[tex]\frac{5}{8}[/tex] = [tex]\frac{5}{8}[/tex]

So we know our values are correct.

Now we calculate for y - x.

20 - 5 = 15

The answer is 15.
