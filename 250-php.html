3. A driver claims that their average net take-home pay in a day is php250. Mr. Tim wants to prove this claim by surveying 15 drivers of the town. The values obtained are Php280, Php 270, Php 250, Php 240, Php 260, Php 280, Php 290, Php 300, Php 230, Php 250, Php 250, Php 260, Php 220, Php 250 and $260. Is there a reason to believe that the average net take-home pay of a driver in the town is Php 250.00? Assume that the net weight is normally distributed with unknown population variance. Find the: 1.) Null Hypothesis: ______________ 1.1.) In symbol, Ho: µ = __________ 2.) Alternative Hypothesis: _________ 2.2.) In symbol, Ha: µ = ___________ 3.) Test Statistic to be used: _______


1.) Null Hypothesis: The average net take-home pay of a driver in the town is Php 250.00.

1.1.) In symbol, Ho: µ = 250

2.) Alternative Hypothesis: The average net take-home pay of a driver in the town is not Php 250.00.

2.2.) In symbol, Ha: µ ≠ 250

3.) Test Statistic to be used: Since the sample size is greater than 30 and the population variance is unknown, we can use the t-test. We can use the formula:

t = (x̄ - µ) / (s / sqrt(n))

where:

x̄ = sample mean

µ = population mean (assumed to be 250)

s = sample standard deviation

n = sample size

We will use a significance level of 0.05.
