24. If x^2+1/x^2=14,find x^3+1/x^3​


CUBING

[tex]\normalsize{\blue{\overline{\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad}}}[/tex]

[tex]\sf{If \: x^{2} + \dfrac{1}{x^{2} } =14, \: find \: x^3 + \dfrac{1}{x^{3}}}[/tex]

[tex]\large\underline{\mathbb{SOLUTION}:}[/tex]

[tex]\boxed{\sf{ \: \: \: x^3 + \dfrac{1}{x^{3}} = 52\: \: \: }}[/tex]

Given that:

[tex]\boxed{\sf{x^{2} + \dfrac{1}{x^{2} } = 14}}[/tex]

On adding 2 from both sides, we get:

[tex]\sf{x^{2} + \dfrac{1}{x^{2} } = 14+2}[/tex][tex]\sf{x^{2} + \dfrac{1}{x^{2} } +2 \times 1 = 16}[/tex][tex]\sf{x^{2} + \dfrac{1}{x^{2} } +2 \times x \times \dfrac{1}{x} = 16}[/tex][tex]\sf{\bigg(x + \dfrac{1}{x }\bigg)^{2} = 16}[/tex][tex]\sf{\bigg(x + \dfrac{1}{x }\bigg)^{2} = 4^{2} }[/tex][tex]\blue{\sf{\bigg(x + \dfrac{1}{x }\bigg)^{2} = 4 }}[/tex]

Now, we have:

[tex]\boxed{\blue{\sf{\bigg(x + \dfrac{1}{x }\bigg)^{2} = 4 }}}[/tex]

On cubing both sides, we get

[tex]\sf{\bigg(x + \dfrac{1}{x }\bigg)^{3} = 4^{3} }[/tex]

[tex]\sf{x^{3} \bigg(\dfrac{1}{x }\bigg)^{3} + 3 \times x \times \dfrac{1}{x} \bigg(x + \dfrac{1}{x} \bigg) = 4^{3} }[/tex]

[tex]\sf{x^{3} + \dfrac{1}{x^{3} } + 3 \times 4 = 64 }[/tex]

[tex]\sf{x^{3} + \dfrac{1}{x^{3} } + 12 = 64 }[/tex]

[tex]\sf{x^{3} + \dfrac{1}{x^{3} }= 64 - 12}[/tex]

[tex]\green{\sf{x^{3} + \dfrac{1}{x^{3} }= 52}}[/tex]

[tex]\sf{ Hence, \: \: \boxed{\green{\sf{x^{3} + \dfrac{1}{x^{3} }= 52}}}}[/tex]
