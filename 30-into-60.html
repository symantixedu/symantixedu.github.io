A.

sin 60° = √3/2

cos 30° = √3/2

Therefore,

sin 60° + cos 30° = √3/2 + √3/2

= √3

B.
To solve this expression, we need to know the values of cot 60° and cos 30°.

We know that cot θ = 1/tan θ, and tan 60° = √3. So, cot 60° = 1/√3 = √3/3.

We also know that cos θ = √(1 - sin^2 θ), and sin 30° = 1/2. So, cos 30° = √(1 - (1/2)^2) = √(3/4) = √3/2.

Therefore, cot 60° + cos 30° = (√3/3) + (√3/2) = (2√3/6) + (3√3/6) = 5√3/6.

Hence, cot 60° + cos 30° = 5√3/6.

C.

We can expand the left-hand side of the equation using FOIL (First, Outer, Inner, Last):

(tan 30° + 1)(tan 30° + 1) = tan 30° x tan 30° + tan 30° x 1 + 1 x tan 30° + 1 x 1

We know that tan 30° = 1/√3, so we can substitute that in:

(tan 30° + 1)(tan 30° + 1) = (1/√3) x (1/√3) + (1/√3) x 1 + 1 x (1/√3) + 1 x 1

Simplifying, we get:

(tan 30° + 1)(tan 30° + 1) = 1/3 + 1/√3 + 1/√3 + 1

Combining like terms, we get:
