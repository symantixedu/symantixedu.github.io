14. 3. What is the result when (x + 5) is multiplied to (x-7)?D. X2 + 2x35A. X- - 2x + 35B. x + 2x + 35C. X 2x - 35​


Answer:

C. x² - 2x - 35

Step-by-step explanation:

(x+5)(x-7)

Using the FOIL method,

Firsts: x(x) = x²Outsides: x(-) = -7xInsides: 5(x) = 5xLasts: 5(-7) = -35

Adding those terms,

x² + (-7x) + 5x + (-35)

= x² - 2x - 35

Hence, we select Option C.
