22. The cab driver charges a fare of $4 for a mile. the meter displays a rate of $7.50 for 2 miles, 11 for 3 miles and so on arithmetic sequence. how much will the meter displays for each of the following distances:4 miles,5 miles and 6 miles?​


Answer:

The meter will show

$14.50 for 4 miles

$18.00 for 5 miles

$21.50 for 6 miles

Step-by-step explanation:

The cab driver charges a fare of $4 for a mile. the meter displays a rate of $7.50 for 2 miles, 11 for 3 miles and so on arithmetic sequence. how much will the meter displays for each of the following distances:4 miles,5 miles and 6 miles?​

Let's identify the given values first.

Let n be the number of miles and a be the fare charged

n = 1, a1 = 4

n = 2, a2 = 7.5

n = 3, a3 = 11

The common difference, d, in this sequence is

d = a2 - a1 = 7.5 - 4 = 3.5

The formula for arithmetic sequence is

[tex]a_n=a_1+(n-1)(d)[/tex]

Substituting the required values

n = 4miles

[tex]a_4=4+(4-1)(3.5)=14.50[/tex]

n = 5miles

[tex]a_5=4+(5-1)(3.5)=18[/tex]

n = 6miles

[tex]a_6=4+(6-1)(3.5)=21.50[/tex]
