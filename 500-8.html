#Progress Check#

Identify the property of real numbers illustrated in each sentence.

Answer:  

1. (32 + 7) + 8 = 7 + (8 + 32) → Associative Property of Addition

2. 2 000 x 0 = 0 → Zero Property of Multiplication

3. 0 + 500 = 500 → Identity Property of Addition

4. (8 x 4) x 10 = (10 x 4) x 8 → Associative Property of Multiplication

5. 15 + 8 + 2 = 8 + 2 + 15 → Associative Property of Addition

6. 50 +-50 = 0 → Inverse Property of Addition

7. 18 x 1 = 18 → Identity Property of Multiplication
