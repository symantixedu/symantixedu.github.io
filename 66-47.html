1. 3. In an isosceles triangle, one base angle measures 57°. Find the measures of the othertwo angles.a. 47, 66b. 56, 66c. 57, 66d. 57, 67​


The answer is C.

An isosceles triangle is a triangle that has two sides of equivalent length and angles. Therefore if one of the base angles is 57°, then the other one should be 57° also.

Since the sum of all the interior angles of a triangle is 180°, we will have the following:

Note: x is the missing angle.

57° + 57° + x = 180°

114° + x = 180°

x = 180° - 114°

x = 66°

Therefore, the two angles are 57° and 66°.

6. 1 Day 1 SeatWork#1: The data shows the height of the students in centimeter. Find the First Quartile of the given data. 55, 47, 38, 66, 56, 64, 44, 63, 39​


To find the first quartile (Q1), we need to first arrange the data in ascending order:

38, 39, 44, 47, 55, 56, 63, 64, 66

Since there are an odd number of data points, the median is the middle value, which is 55.

Next, we split the data into two halves:

38, 39, 44, 47, | 55 | 56, 63, 64, 66

The lower half of the data has 4 values, so we find the median of this half to get the first quartile:

Q1 = median(38, 39, 44, 47) = 41.5

Therefore, the first quartile of the given data is 41.5 cm.
