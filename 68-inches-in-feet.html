19. a scale model of a triangle tent entrance has a base of 12 inches and a perimeter of 42 inches. Its actual base is 68 inches. What is the perimeter of the actual tent entrance?​


Step-by-step explanation:

A rectangle has a perimeter of 40 inches. It is 3 times as long as it is wide.

Answer:

The perimeter of the actual tent entrance is 252 inches.

Step-by-step explanation:

We know that the scale factor of a model is the ratio of the measurements of the model to the measurements of the actual object. To find the scale factor, we can divide the measurement of the model by the measurement of the actual object.

In this case, the scale factor is:

12 inches / 68 inches = 1/6

We can use this scale factor to find the actual measurements of the tent entrance.

To find the perimeter of the actual tent entrance, we can use the formula:

Perimeter of actual = Perimeter of model x Scale factor

So,

Perimeter of actual = 42 inches x (1/6)Perimeter of actual = 42 inches x (1/6) = 252 inches

Therefore, the perimeter of the actual tent entrance is 252 inches.

29. the width of a rectangle is 8 inches shorter than its length. if the sum of the width and the length is 68 inches, find the width and length representation:equations:


Representation:

Let the width of the rectangle be x

⇒ Width = x

⇒ Length = x + 8


Form the equation:

The sum of the width and the length is 68 inches

⇒ x + (x + 8) = 68


Solve x:

x + (x + 8) = 68

x + x + 8 = 68

2x + 8 = 68

2x = 60

x = 30


Find the length and width:

Width = x = 30 inches

Length = x + 8 = 30 + 8 = 38 inches


Answer: The width is 30 inches and the length is 38 inches
