Answer:

This problem involves finding the coordinates of a point that lies on a curve whose tangent is parallel to a given line. The given curve is y = x^2 - 5x + 5, and the given line is 2y = 4x + 1.

To find the point, we need to first find the x-coordinate of the point where the tangent to the curve is parallel to the given line. Since the tangent to the curve has the same slope as the given line, we can equate the derivatives of the curve and the line:

dy/dx = 2x - 5 = 2 (slope of tangent)

2y = 4x + 1 (given line)

Setting the slopes equal to each other, we get:

2x - 5 = 2

2x = 7

x = 7/2
