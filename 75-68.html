6. Find range, and mean absolute deviation of the given data : 88, 75, 91, 68, 95, 79, 86, 99


Answer:

Hope this helps

To find the range, we need to find the difference between the largest and smallest values in the dataset:

Range = Largest Value - Smallest Value

Range = 99 - 68 = 31

So the range of the given data is 31.

To find the mean absolute deviation (MAD), we need to follow these steps:

Find the mean of the data.
Find the absolute deviation of each value from the mean.
Find the mean of these absolute deviations.
Step 1: Find the mean

Mean = (88 + 75 + 91 + 68 + 95 + 79 + 86 + 99) / 8
Mean = 83.125

Step 2: Find the absolute deviation of each value from the mean

Absolute deviation of 88 = |88 - 83.125| = 4.875
Absolute deviation of 75 = |75 - 83.125| = 8.125
Absolute deviation of 91 = |91 - 83.125| = 7.875
Absolute deviation of 68 = |68 - 83.125| = 15.125
Absolute deviation of 95 = |95 - 83.125| = 11.875
Absolute deviation of 79 = |79 - 83.125| = 4.125
Absolute deviation of 86 = |86 - 83.125| = 2.875
Absolute deviation of 99 = |99 - 83.125| = 15.875

Step 3: Find the mean of these absolute deviations

MAD = (4.875 + 8.125 + 7.875 + 15.125 + 11.875 + 4.125 + 2.875 + 15.875) / 8
MAD = 8.4375

So the range of the given data is 31 and the mean absolute deviation is 8.4375.
