Difference between rows and columns in excel

Microsoft Excel is a spreadsheet application developed by Microsoft Corporation. When we say spreadsheet, it is a type of application that performs calculations and computations, specifically accounting computations with conditions (like IF statements), loops (while statements) and macros (special codes, like programming codes).  

One spreadsheet file can contain different worksheets inside. One worksheet can have collections of rows and columns. A row is a part of the spreadsheet which is presented horizontally. Normally, row has a number header. On the other hand, column is a part of the spreadsheet that is presented vertically, and is represented by letters. The intersection of columns and rows is called cell. Cells are needed to create formulas in excel.
