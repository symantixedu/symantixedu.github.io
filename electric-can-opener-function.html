1. what is the function of electric can opener?


The function of an electric can opener is to open can easily. Electric Can Opener is more convenient to use than a manual can opener, but the process is very much the same. The difference is that electric can opener has a physical motion of turning the wheel that spins the can automatically, using the electricity. In using Manual Can Opener, you need to use your strength and arms to open the can. manual can opener function by placing the metal tooth on the lip of the can, to align the wheel to the rim of the can. You need to grip the two handle using one arm for a secure fit and the other arm to start turning handle to rotate the wheel. That is the difference between electric and manual can opener.
