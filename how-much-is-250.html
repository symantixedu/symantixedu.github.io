1. how much alcahol is in 250 ml of 40% solution


Answer:

Explanation:

V1 (solute volume) = ? (in mL)

V (volume of the solution) = 250 mL

%v/v = v1/v * 100

40*250 = 100

v1 = 10000/100

v1 = 100 ml alcohol. = FINAL ANSWER

7. how much work is done in lifting an object that weighs 250 N to a height of 4m


[tex]\tt{\huge{\blue{Explanation:}}}[/tex]

The work done by an object exerting a force F at a distance d is given by

[tex]\boxed{W = Fd}[/tex]

where:

W = work

F = force

d = distance

[tex]\tt{\huge{\red{Solution:}}}[/tex]

Solving for W

W = Fd

W = (250 N)(4 m)

[tex]\boxed{W = \text{1,000 J}}[/tex]

Therefore, the work done is 1,000 J.
