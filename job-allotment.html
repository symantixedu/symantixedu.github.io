21. A family budget provides an expenditures of Php5,100.00 per month for food. If the amount allotted for meat is Php300 more than that of milk and if the allotment for fruits is twice as much as that for meat and milk, find the amount allotted for milk. *


Answer:

The price alloted for the milk is Php 700.

Step-by-step explanation:

This is a worded type of problem. From the given words, we need to create equation/s with unknown variables.

Given:

Php 5,100 total expenditure for food

Let variables for items that are unknown

Let x = price of milk

Let x + 300 = price of meat

Let 2 [(x + x + 300] = price of fruit

Solving for the value of x, we equate the sum of the prices of milk, meat and fruit to 5, 100

5,100 = x + (x+300) + 2 [x+x+300}

5,100 = x + x + 300 + 2 (2x+300)

5, 100 = 2x + 300 + 4x + 600

5,100 = 6x + 900

6x = 5,100 - 900

6x = 4200

x = 4200/6

x = 700

Soving, we have

price of milk: x = Php 700

price of meat: x + 300 = Php 700 + Php 300 = Php 1,000

price of fruit: 2 [(x + x + 300] = 2 (1400+300) = Php 3,400

Checking:

Price of milk + Price of meat + Price of fruit = Total expenditure for food

Php 700 + Php 1,000 + Php 3, 400 = Php 5,100
