1. Outlining format that uses roman numerals, letters, and numbers​


Answer:

Begin with a Roman Numeral for the Topic. After each Roman numeral, letter, or number put a period. Begin all lines with an upper case letter. Line up every letter or numeral under the one above. The main points of your topic will look like an abbreviated topic sentence for each paragraph. The next Roman numeral, or “II” will be for first body paragraph. The third “III” for the second body, and so on. When the outline is complete, each Roman numeral should equal the number of paragraphs in the paper.
