16. ordinal numbers and ordinal shape​


Answer:

ordinal numbers

1st, 2nd, 3rd and so on

ordinal shape

1st shape circle

2nd shape square

3rd shape Triangle

Explanation

Ordinal numbers are numbers that indicate the position or order of an object in a sequence.

For example, "first," "second," "third," and so on, are ordinal numbers.

Ordinal shape is a term used in mathematics to describe the geometric shape of an object that has a specific ordinal position in a sequence.

For example, the first shape in a sequence might be a circle, the second might be a square, the third might be a triangle, and so on. The ordinal shape of an object is often used in the study of patterns and sequences in mathematics.
