Answer:

Vitamin D is a group of fat-soluble secosteroids responsible for increasing intestinal absorption of calcium, magnesium, and phosphate, and many other biological effects. In humans, the most important compounds in this group are vitamin D₃ and vitamin D₂.

6. Ano ang tungkulin ng iron? Saan pinanggalingan ang iron? Ano ang tungkulin ng zinc? Saan pinanggalingan ang zinc? Vitamin c ano ang tungkulin? Vitamin c saan pinang galiganVitamin a ang ang tungkulin? Vitamin a saan pinanggalingan? Iodine ano ang tungkulin? Lodine saan pinanggalingan? ​


Answer

Ano ang tungkulin ng iron?

Ang Iron ay importante para sa dugo. Halos 70% ng Iron ay matatagpuan sa Red Blood Cells na ang tawag ay Hemoglobin. Mayroon din Iron sa Muscle cells na ang tawag ay Myoglobin.

Saan pinanggalingan ang iron?

Ang pangunahing pinagkukunan ng katawan ng Iron ay sa pagkain ng mga karne ng hayop.

Ano ang tungkulin ng zinc?

Ang Sinc ay tumutulong sa ating Immune System at Metabolism.

Saan pinanggalingan ang zinc?

Maggandang pagkunan ng Zinc ay karne ng hayop at ng itlog.

Vitamin c ano ang tungkulin?

Ang tungkulin ng Vitamin C sa ating katawan ay maging antioxidant. Tumutulong ito na pangalagaan at proteksyonan ang mga cells sa pinsala dulot ng Free radicals.

Vitamin c saan pinanggalingan?

Ang pinakamayamang pagkukunan ng Vitamin C ay sa nga prutas tulad ng Oranges, Dayap, Mandarin, Pomelo, at iba pang Citrus Fruit.

Vitamin a ang ang tungkulin?

Ang Vitamin A ay tumutulong sa paglaki ng ating katawan at sa pangangalaga ng ating paningin

Vitamin a saan pinanggalingan?

Ang Vitamin A ay sagana sa mga Berdeng Gulay, Prutas, at matatagpuan din sa mga Cereals.

Iodine ano ang tungkulin?

Ang Iodine ay kailangan ng katawan upang makagawa ng Thyroid Hormones. Ang mga Hormones na ito ang kumokontrol sa metabolismo ng ating katawan. Kailangan din ng mga bata ang thyroid hormones para lumaki silang malakas ang mga buto at aktibo ang paglaki ng utak.

Iodine saan pinanggalingan? ​

Ang Iodine ay sa gana sa mga produktong dairy tulad ng Keso at Gatas. matatagpuan din ang Iodine sa mga lamang-dagat.
