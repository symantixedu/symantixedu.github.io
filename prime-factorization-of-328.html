29. X+y=100 and 2x+4y=328


Hello !!

You has the equation x + y = 100.Now you pass [y] to the other side of equality, negative.

x + y = 100

x = 100 - y

Having done the previous process you replace [x] with (100 - y) in the other equation.

2x + 4y = 328

2(100 - y) + 4y = 328

200 - 2y + 4y = 328

200 + 2y = 328

2y = 328 - 200

2y = 128

y = 128/2

y = 64

Having found the value of [y], you substitute the value of [y] in any of the equations and find the value of [x].

x + y = 100

x + 64 = 100

x = 100 - 64

x = 36

Final result : The solution for [x] is 36 and the solution for [y] is 64.
