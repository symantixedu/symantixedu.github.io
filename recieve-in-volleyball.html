13. In a class election,mark recieved twice as many votes as ariane,ariane recieved twice as many votes as andrew.there were 35 votes altogether.how many votes did andrew recieve?


Answer:

Let the number of votes Andrew received be x.

Then, Ariane received 2x votes, and Mark received 2(2x) = 4x votes.

The total number of votes is x + 2x + 4x = 7x.

We know that the total number of votes is 35, so 7x = 35.

Thus, x = 5, and Andrew received 5 votes.

Let's start by assigning variables to represent the number of votes each candidate received:

Let A be the number of votes that Andrew received.

Then Ariane received twice as many votes as Andrew, which means Ariane received 2A votes.

Mark received twice as many votes as Ariane, which means Mark received 2(2A) = 4A votes.

The problem tells us that there were 35 votes altogether, so we can write an equation:

A + 2A + 4A = 35

Simplifying the left side of the equation, we get:

7A = 35

Dividing both sides by 7, we find:

A = 5

Therefore, Andrew received 5 votes.
