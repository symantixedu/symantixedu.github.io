Word Problem Involving Quadratic Equation

Quadratic equation refers to an equation of the second degree. This means that the equation contains at least one term that is squared. The standard form is ax¬≤ + bx + c = 0 with a, b, and c being constants, or numerical coefficients, and x is an unknown variable.

Going back to the word problem above, it is asking for the dimensions of the pool. To find the answers, you have to identify the given facts and the equation.

Given:

length - 2 yards more than 2 times the width

area of the pool - 144 square yards

Equation:

Since the area of a rectangular pool is the product of its length and width, we have to use that formula to express the equation. Let "x" be the width.

length √ó width = area

(2 + 2x) x = 144

Solution:

(2 + 2x) x = 144

2x + 2x¬≤ = 144 (Equate to 0)

2x¬≤ + 2x - 144 = 0 (Factor)

(2x - 16) (x + 9) (Isolate the variable)

2x - 16 = 0

2x = 16

[tex] \frac{2x}{2} = \frac{16}{2} [/tex]

x = 8

x + 9 = 0

x = -9

As you notice, there are two possible values of "x". Since the width of a rectangular pool should be a positive number, choose the positive value which is 8. Thus, the width is 8 yards.

Now that you already know the width, substitute its value to find the length.

(2 + 2x)

= [2 + 2 (8)]

= 2 + 16

= 18

The length is 18 yards.

Answer:

The dimensions of the pool is 18 yards by 8 yards.
