The central tendency for number of laps above are (mean = 13), (median = 12), mode (8 & 15).

Step-by-step explanation:

Mean : 13

Mean = (sum of terms) / (number of terms)

Mean = (20 + 12 + 13 + 8 +26 + 7 + 8 + 9 + 11 + 12 + 15) / (11)

Mean = 141 / 11

Mean = 12.81

Mean = 13

Median : 12

If the number of observations is odd, the number in the centre or middle of the list is the median. This can be found by taking the value of the (n+1)/2 -th term, where n is the number of observations.

The list of number would be :

Median = number in the middle of 7, 8, 8, 9, 11, 12, 12, 13, 15, 20, 26

Median = (7, 8, 8, 9, 11,) 12, (12, 13, 15, 20, 26)

Median = 12

Mode : 15

Mode is the most frequent number on the list/sequence

Mode = 7, 8, 8, 9, 11, 12, 12, 13, 15, 20, 26

Mode = 7, (8, 8,) 9, 11, (12, 12,) 13, 15, 20, 26

Mode = 8 & 15

If there are two numbers that appear most often (and the same number of times) then the data has two modes. This is called bimodal. If there are more than 2 then the data would be called multimodal. If all the numbers appear the same number of times, then the data set has no modes.
