2. You are to pick three singers for the Superbowl, one to sing before the game, one to sing at the end of the game, and one to sing at halftime. If there are 12 applicants, how many different possibilities are there if each singer can only sing once?


Solution:

So we use the Formula of Combination for 12C3:

nCk = n!/k(n-k)!

Let n = 12 and k = 3

12C3 = 12!/3!(12-3)!
12C3 = 12!/3!(9)!
12C3 = 220

And for the three singer, we have,

3! = 6 

Then, we multiplied it, 
220 x 6 
= 1320

Therefore, there are 1320 different possibilities if each singer can only sing once.Answer: 7,920
.
We can use the formula for permutation of n objects taken r at a time.
n = 12
r = 3
P(n,r) = n! / (n-r)!
P(12,3) = 12! / (12-3)!
P(12,3) = 12x11x10x9! / 9!
(Just cancel the 9! on the numerator and denominator for easier solving.)
P(12,3) = 12x11x10
P(12,3) = 1,320
.
It's not yet the final answer.
Now, we can still arrange the 3 singers.
3! = 3×2×1 = 6
1,320 × 6 = 7,920
