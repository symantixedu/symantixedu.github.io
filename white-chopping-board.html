Chop is to chopping board as drain is to strainer.

1This is an example of a function / object analogy since the first word is the function of the second word.

2. Cry is to whine as cute is to pretty.

This is an example of a synonym analogy because they mean the same thing.

3. Scientist is to microscope as engineer is to blueprint is another example to note.

This is an example of an worker / tool analogy because the first word uses the second word in their job.

4. Flower is to garden as artifacts is to museum.

This is an example of a part / whole analogy since a flower is a part of a garden, while artifacts are found in a museum.

5. Nun is to pray as tailor is to sew.

This is an example of an object / purpose analogy, but it is probably better to say it is worker / purpose since this includes a person.
